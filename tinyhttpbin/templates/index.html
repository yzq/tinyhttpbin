tinyhttpbin: HTTP Request & Response Service

  • /ip Returns Origin IP.
  • /user-agent Returns user-agent.
  • /headers Returns header dict.
  • /get Returns GET data
  • /cookie/set?name=value Sets a simple cookie.
  • /cookies/set Sets one or more cookies.
  • /response-headers Return a set of response headers from the query string.
  • /redirect-to 302 Redirects to the URL.
  • /delay/:n Delays responding for min(n, 10) seconds.
  • /images/jpeg Return a jpeg image.
  • /xml Return a xml file.
  • /robot.txt Returns some robots.txt rules.
  • /deny Denied by robot.txt file
  • /cache Returns 200 unless an If-Modified-Since or If-None-Match header is provided, when it returns a 304..
  • /cache/n Sets a Cache-Control header for n seconds.
